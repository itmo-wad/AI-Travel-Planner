# Presentation content

## Slide 1 — Title

AI Travel Planner  
AI-powered personalized trip planning app

## Slide 2 — Problem

Planning a trip is time-consuming and chaotic. Users have to search through many sources, compare attractions, check timing, estimate distances and fit everything into a realistic route.

## Slide 3 — Solution

A web application where users enter destination, dates, budget and interests. AI generates a structured day-by-day itinerary with activities, timing, budget notes and alternatives.

## Slide 4 — Target users

- students
- tourists
- groups of friends
- people with limited time for planning
- users who need personalized recommendations

## Slide 5 — User stories

Main:
As a user, I want to enter my destination, dates, budget and interests, so that I can receive a personalized itinerary.

Additional:
- save generated trips
- edit itinerary activities
- regenerate one day
- get budget-friendly suggestions
- get bad-weather alternatives

## Slide 6 — User flow

Open app → create trip → enter preferences → generate itinerary → view plan → edit/save trip

## Slide 7 — UI mockups

Screens:
- create trip form
- generated itinerary
- saved trips dashboard
- trip details page

## Slide 8 — Architecture

Frontend → Backend API → AI service → structured itinerary  
Backend also communicates with database for users and saved trips.

## Slide 9 — Database

Tables:

- trip_preferences
- itinerary_days
- itinerary_items
